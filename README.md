# Salient NoCap Child

This child theme contains a full front-page redesign for NoCap Barbers.

## Activate
1. WordPress Admin -> Appearance -> Themes
2. Activate **Salient NoCap Child**

## Roll Back
1. WordPress Admin -> Appearance -> Themes
2. Activate **Salient**

## Files
- `style.css`: Child theme header + all redesign styling.
- `functions.php`: Front-page body class, fonts, animation script enqueue, skip link hook, fallback schema.
- `front-page.php`: Fully custom homepage layout and section structure.
- `assets/js/home-modern.js`: Scroll reveal + subtle hero parallax animation.

## Notes
- Parent theme files are untouched.
- Navigation anchor links are preserved: `#uber-uns`, `#service`, `#gallerie`, `#team`, `#kontakt`.
